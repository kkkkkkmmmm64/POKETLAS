# Poketlas

> Find real-world geographic names that exactly match official Pokémon names in any supported language.

## Overview

Poketlas is an open-source project that discovers real-world geographic names matching official Pokémon names across all officially supported languages.

The project uses official Pokémon name data together with the GeoNames geographical database to identify exact matches and visualize them on a world map.

## Goals

- 🌍 Search real-world geographic names
- 🎮 Match official Pokémon names
- 🌐 Support every official Pokémon language
- 📊 Export CSV datasets
- 🗺️ Export GeoJSON for mapping
- 📈 Generate statistics and reports
